## What Split Screen Launcher Does

Split Screen Launcher is an Android productivity tool for anyone who relies on split screen and wants a quicker path back to the same two apps. Rather than rebuilding your multitasking layout each time, you can store app pairs and launch them again with one tap.

The app keeps things minimal and easy to use while still covering current Android releases, including Android 12L, Android 15, and Android 16. It does not ask for Accessibility permission, so setup stays straightforward and you can get started immediately.
